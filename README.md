## Visão Geral

> *O objetivo deste projeto é implementar um sistema para restaurantes de cadastro de cardápio e contas de mesas. Este sistema de cadastro deve ser operado através de um terminal (prompt/shell) que funciona como um interpretador de comandos de texto, ou seja, o sistema vai ficar constantemente checando por entradas de texto (string) e interpretando o que for recebido a partir de uma lista de comandos que o sistema deve ser capaz de executar.*

## Requisitos

- **Java 8** ou superior.
- **Mars 4.5**.

## Configurações importantes

- Ative `Assemble all files in directory` para fazer a build do projeto.
